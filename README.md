# Judoss AI — Prompt Operating System

Judoss AI is a browser-based prompt-workflow studio that transforms a structured request into a reusable execution pack. It supports prompt creation and response workflows across planning, coding, image, and debugging modes.

## Features

- Create or reply workflow modes
- Structured inputs for objectives, target platform, constraints, and assets
- Model-profile and temperature controls
- JSON, Markdown, and prompt-text output formats
- Alignment preview, clipboard copy, and export actions

## Run locally

Open `index.html` in a browser.

## Tech stack

HTML, CSS, and JavaScript.
